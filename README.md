# Hand-Tracking-Virtual-Keyboard

## Overview

A real-time hand-tracking virtual keyboard that replicates a physical Korean/English keyboard layout and provides integrated web search functionality using OpenCV and MediaPipe.

## Features

- Real-time hand tracking
- Physical Korean/English keyboard layout
- Hangul composition
- Shift support
- Integrated Google Search
- Delete and Space keys

## Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
- Pillow
- Jamo

## Installation

```bash
pip install -r requirements.txt
python main.py
```
